# Netflix SQL Analysis

## 📌 Project Overview

This project analyzes a Netflix dataset using MySQL. It includes data cleaning, SQL queries, business insights, and data analysis techniques.

## 📂 Dataset

Netflix Movies and TV Shows Dataset

## 🛠 Tools Used

- MySQL
- SQL

## 📚 SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- COUNT()
- AVG()
- MAX()
- CASE WHEN
- CAST()
- REPLACE()
- STR_TO_DATE()
- DATE_SUB()
- Subqueries
- SELF JOIN
- UNION

## 📈 Business Questions Solved

- Movies vs TV Shows
- Missing value handling
- Most common rating
- Top countries by content
- Longest movies
- Average movie duration
- Content released by year
- Recent additions
- Duration categorization
- Director analysis

## 🎯 Key Insights

- The United States has the highest number of content titles.
- TV-MA is the most common rating.
- The average movie duration is approximately 101 minutes.
- Movies are slightly more common than TV Shows.
